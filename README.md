# 🟩 V-R-CRAFT | Developer Portal

![V-R-CRAFT Banner](assets/image/gun_v4.png)

> **"Voxel, Script API, and AI Convergence."**

Minecraft（Voxel）のアドオン開発を核とし、高度な **Script API** と **AI（Gemini）** を融合させた次世代の制作環境と成果物を公開する、開発者「Neo」の公式ポータルサイトです。

## 🌟 Key Features

- **Advanced Addons**: 高精度な弾道計算（3D GUN V4）や Script API を駆使したエフェクト（THUNDER SWORD）など、技術的な限界を追求。
- **AI Integration**: 最新の **Gemini 1.5 Flash** を搭載したサポートアシスタント。
- **Modern UI**: Tailwind CSS とグラスモーフィズムを採用した、洗練されたサイバーパンク・デザイン。
- **Open Architecture**: `data.js` による一元管理、ダークモード永続化、GitHub API との動的連携。

## 🛠 Tech Stack

- **Frontend**: HTML5, Tailwind CSS, JavaScript (ES6+)
- **AI**: Google Gemini API (1.5 Flash)
- **Scripting**: Minecraft Script API (V2+)
- **Management**: Git / GitHub

## 🚀 Getting Started

### Local Setup
1. このリポジトリをクローンします。
   ```bash
   git clone https://github.com/xakryukib-star/VR-CRAFT-Portal.git
   ```
2. `index.html` をブラウザで開きます。

### AI Assistant Setup
1. サイト内の **ASSISTANT** ページへ移動します。
2. `CONFIG API KEY` から、あなたの [Google AI Studio](https://aistudio.google.com/) キーを設定してください。

## 📁 Structure

- `index.html`: ホームページ
- `download.html`: アドオン・ライブラリのダウンロードセンター
- `assistant.html`: AI サポート・チャット
- `discord.html`: コミュニティ・リンク
- `data.js`: アドオン・メタデータの一元管理
- `assets/`: 画像、スクリプト、スタイル等のアセット
