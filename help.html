Frequently Asked Questions

#01 如何にしてアドオンを導入するか?

ダウンロードした `.mcaddon` または `.mcpack` ファイルをダブルクリック（または実行）すると、Minecraftが自動的に起動しインポートが開始されます。

#02 Script APIが動作しない場合は?

ワールド設定の「実験」タブにある「ベータ API (Beta APIs)」がオンになっているか確認してください。これがオフだとスクリプトは読み込まれません。

#03 3Dモデルが表示されない?

リソースパックが行動パック（ビヘイビア）よりも上位に配置されているか確認してください。また、デバイスのメモリ不足によりモデルがスキップされる場合があります。

#04 二次配布は可能ですか?

V-R-Craftの全ての成果物は、クレジットの有無に関わらず無断での二次配布を禁止しています。紹介の際は当サイトのリンクを共有してください。
